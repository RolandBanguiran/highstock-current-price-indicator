# Highstock Current Price Indicator
Using this plugin will display current price indicator in Highstock.

## Default Options:
```javascript
yAxis: {
  currentPriceIndicator: {
    backgroundColor: '#000000',
    borderColor: '#000000',
    enabled: true,
    lineColor: '#000000',
    lineDashStyle: 'Solid',
    lineOpacity: 0.8,
    style: {
      color: '#ffffff',
      fontSize: '11px'
    },
    x: 0, //offset
    y: 0, //offset
    zIndex: 7,
	labelFormatter: null		//advanced price to text formatter
  }
}
```
